# 🆘 Support for ACLAS Neuro-Edu

Thank you for using the Neuro-Edu framework. We want to ensure you have a smooth experience.

## 🔍 How to Get Help

### 1. Documentation
Before opening an issue, please check our:
- [README.md](README.md)
- [Technical Whitepaper (DOI: 10.5281/zenodo.19743807)](https://doi.org/10.5281/zenodo.19743807)
- [Hugging Face Space Documentation](https://huggingface.co/spaces/ACLASCollege/aclas-neuro-dashboard)

### 2. GitHub Discussions
For "How-to" questions, architecture advice, or showcasing your work, please use the **[GitHub Discussions]** tab (if enabled) or open an issue with the `question` label.

### 3. Issue Tracker
For bug reports and feature requests, please search the **[Issue Tracker](https://github.com/aclascollege/neuro-edu/issues)** first. If your problem is unique, use our templates to open a new one.

---

## 🏛️ Professional & Institutional Support
For institutional partnerships, academic collaboration, or certification inquiries, please contact ACLAS directly:

- **Official Website**: [https://aclas.college](https://aclas.college)
- **Email**: [info@aclas.college](mailto:info@aclas.college)

## 🛡️ Security Vulnerabilities
Please do **not** report security vulnerabilities via public issues. See our [SECURITY.md](SECURITY.md) for the private disclosure process.
