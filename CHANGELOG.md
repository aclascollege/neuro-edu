## [3.0.0] — 2026-04-23

### 🎉 Initial Public Release

The first open-source release of the **ACLAS Neuro-Edu SDK** under MIT License.

### Added
- **On-Device Neural Kernel** — Pure NumPy MLP with real backpropagation (He init, ReLU, Sigmoid, SGD).
- **Multi-Agent Social Learning** — 20+ autonomous agents exchanging knowledge via a real-time message bus.
- **Cognitive Entropy Analytics** — Thermodynamic-inspired Shannon entropy tracking for classroom dynamics.
- **5-Dimensional Skill Matrix** — Logic / Math / Language / Memory / Creative agent profiles.
- **Federated Training** — Each agent trains its own MLP on a shared dataset with observable loss curves.
- **3D Neural Nebula** — WebGL (Three.js) visualization of agent states in latent space.
- **Advanced Scoring** — GPA (4.0), CAS Score, Retention Rate, Shannon Diversity Index, Dropout Risk.
- **Knowledge Graph** — D3.js-powered live concept relation map.
- **CLI Tool** — `python cli.py simulate / train / report` for fully scriptable operation.
- **26 pytest Tests** — Industrial-grade coverage across model, agent, and engine layers.
- **GitHub Pages Demo** — Interactive static demo with MockEngine at [aclascollege.github.io/neuro-edu](https://aclascollege.github.io/neuro-edu/).
- **Multi-language READMEs** — English, French, Spanish, Russian, German, Traditional Chinese, Japanese, Korean.
- **Community health files** — CODE_OF_CONDUCT, CONTRIBUTING, SECURITY, Issue & PR templates.
