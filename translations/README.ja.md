## 🌌 ACLAS Neuro-Edu とは？

**ACLAS Neuro-Edu** は、研究グレードのオープンソース・マルチエージェント認知シミュレーションフレームワークです。外部の AI API に頼ることなく、**オンデバイス・ニューラルネットワーク**を使用して、知識習得のプロセスをモデル化します。

> *"私たちは GPT をラップしません。第一原理から認知をモデル化します。"*

### 🌟 主な機能

- 🧠 **オンデバイス・ニューラルカーネル**：純粋な NumPy による MLP（バックプロパゲーション対応）。
- 🤗 **Hugging Face 統合**：共同研究のために認知モデルの重みをエクスポート可能。
- 📡 **ソーシャル・ラーニング**：20以上のエージェントがリアルタイムで知識を交換。
- 🛡️ **産業グレードの CI/CD**：自動化されたセキュリティ監査、依存関係チェック、およびベンチマーク。
- 📊 **認知エントロピー分析**：学習の「秩序」と「カオス」をモニタリング。
- 📈 **高度なスコアリング**：GPA、CAS スコア、定着率、中退リスク分析。

## 🔌 プラガブルなLLM認知エンジン

Neuro-Eduには、第一原理に基づいたミニマリストのニューラルネットワーク（`TinyCognitionModel`）がネイティブで搭載されていますが、私たちのフレームワークは高度な拡張性を持つように設計されています。**Ollama**や**vLLM**をシームレスに接続し、世界をリードするオープンソースモデルを使用して、各AI生徒の内なる声と思考プロセスを駆動することができます！

マルチエージェントサンドボックス内の認知エンジンとして、以下のオープンウェイトモデルのテストを強くお勧めします:
- 🦙 **Llama-3 & Llama-3.1** (Meta)
- 🤖 **Qwen 2.5** (Alibaba Cloud) - *私たちの`agent.py`のデフォルト設定*
- 🐋 **DeepSeek-V3 / DeepSeek-Coder** (DeepSeek)
- 💎 **Gemma-2** (Google)
- 🔬 **Phi-3** (Microsoft)
- 🌪️ **Mistral / Mixtral** (Mistral AI)

独自の大規模モデルアリーナを構築し、さまざまなモデルが教育的刺激にどのように反応するかを観察してください！
