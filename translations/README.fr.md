## 🌌 Qu'est-ce que ACLAS Neuro-Edu ?

**ACLAS Neuro-Edu** est un cadre de simulation cognitive multi-agent open-source de niveau recherche. Il modélise le processus d'acquisition des connaissances à l'aide de **réseaux de neurones locaux**, de la **sémantique vectorielle** et de l'**analyse de l'entropie**.

> *"Nous n'utilisons pas d'enveloppes GPT. Nous modélisons la cognition à partir des principes fondamentaux."*

### 🌟 Caractéristiques Clés

- 🧠 **Noyau Neuronal Local** : MLP en pur NumPy avec rétropropagation réelle.
- 🤗 **Intégration Hugging Face** : Exportation des poids des modèles cognitifs.
- 📡 **Apprentissage Social** : Plus de 20 agents autonomes échangeant des connaissances.
- 🛡️ **CI/CD de classe industrielle** : Audit de sécurité, vérification des dépendances et benchmarks.
- 📊 **Analyse de l'Entropie** : Suivi de l'ordre sémantique vs chaos en temps réel.
- 📈 **Métriques Avancées** : GPA, score CAS, taux de rétention et risque de décrochage.

## 🚀 Démarrage Rapide

```bash
git clone https://github.com/aclascollege/neuro-edu.git
cd neuro-edu
pip install -r requirements.txt
python main.py
```

Ouvrez **http://localhost:8000** dans votre navigateur.
