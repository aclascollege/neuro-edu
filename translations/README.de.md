## 🌌 Was ist ACLAS Neuro-Edu?

**ACLAS Neuro-Edu** ist ein Open-Source-Framework zur Simulation kognitiver Multi-Agenten-Prozesse. Es modelliert den Wissenserwerb mithilfe von **On-Device-Neuronalen-Netzen**, **Vektorsemanitk** und **Entropieanalyse**.

> *"Wir nutzen keine GPT-Wrapper. Wir modellieren Kognition nach grundlegenden Prinzipien."*

### 🌟 Hauptmerkmale

- 🧠 **On-Device Neural Kernel**: MLP in purem NumPy mit echtem Backpropagation.
- 🤗 **Hugging Face Integration** : Export von kognitiven Modellgewichten.
- 📡 **Soziales Lernen**: Über 20 autonome Agenten tauschen Wissen in Echtzeit aus.
- 🛡️ **Industrielle CI/CD** : Sicherheitsaudit, Abhängigkeitsprüfung und Performance-Benchmarks.
- 📈 **Erweiterte Metriken**: GPA, CAS-Score, Retentionsrate und Dropout-Risiko.

## 🚀 Schnellstart

```bash
git clone https://github.com/aclascollege/neuro-edu.git
cd neuro-edu
pip install -r requirements.txt
python main.py
```

Öffnen Sie **http://localhost:8000** im Browser.
