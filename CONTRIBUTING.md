# Contributing to ACLAS Neuro-Edu

First off, thank you for considering contributing to ACLAS Neuro-Edu! It's people like you that make the open-source education community such a great place.

## Code of Conduct

By participating in this project, you are expected to uphold our [Code of Conduct](CODE_OF_CONDUCT.md).

## How Can I Contribute?

### Reporting Bugs
- Use the [Bug Report Template](.github/ISSUE_TEMPLATE/bug_report.md).
- Describe the expected vs. actual behavior.
- Provide a minimal reproducible example if possible.

### Suggesting Enhancements
- Use the [Feature Request Template](.github/ISSUE_TEMPLATE/feature_request.md).
- Explain why this enhancement would be useful to the ACLAS community.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes (`pytest tests/`).
4. Make sure your code lints.
5. Issue that pull request!

## Development Setup

```bash
pip install -r requirements.txt
pip install pytest
pytest tests/
```
